Tugas Karya Akhir (TKA) · Kelas 9 · Mata Pelajaran Sejarah & TIK

Vanilla JavaScript CSS3 Glassmorphism Web Audio API LocalStorage Bento Grid Plus Jakarta Sans

🎯 Tujuan Proyek

Historia System adalah game kuis edukasi berbasis web yang dirancang untuk membantu siswa kelas 9 memahami dan mengingat peristiwa penting dalam Sejarah Indonesia secara interaktif dan menyenangkan.

⚙️ Fitur Utama

🎮 Game kuis interaktif dengan 3 nyawa
⏱️ Timer dinamis dengan efek visual
🔊 Audio sintetis via Web Audio API
✍️ Efek typewriter pada pertanyaan
📳 Screen shake saat jawaban salah
🏆 Leaderboard permanen (localStorage)
🌙 Dark mode dengan glassmorphism

📚 Materi Soal

Bank soal mencakup berbagai era sejarah Indonesia: Kerajaan Sriwijaya dan Majapahit, Era Kolonialisme VOC dan Belanda, Kebangkitan Nasional, Proklamasi Kemerdekaan 1945, Orde Lama dan Orde Baru, serta peristiwa reformasi.

💡 Teknologi

Dibangun 100% dengan teknologi web standar tanpa library eksternal. Menggunakan HTML5, CSS3 dengan custom properties dan animasi keyframe, serta Vanilla JavaScript murni untuk semua logika permainan.
